# Internal Dashboard

Internal analytics dashboard for NovaTech Solutions.

## Tech Stack
- Node.js + Express
- React Frontend
- PostgreSQL Database
- Redis Cache

## Setup
1. Clone the repo
2. Run `npm install`
3. Configure `.env` file
4. Run `npm run dev`
